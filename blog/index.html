    2024軟體工程師體驗營 六角學院
    六角學院2024軟體工程師體驗營心得

    這是 799 就能拿到的東西嗎????


        大概是我參加六角學院軟體工程師體驗營最大的想法。參加體驗營其實算是一個很大的挑戰，因為必須在有限的時間內產出產值，大量的內容，在短時間內要全部完成，其實對在職的新手小白並沒有很友善，要在短短的一個月內全部學完其實有相當大的困難。

      體驗營的內容主要有幾個部分:

    1. 60 個小時的影音課程:
        這部分囊括 HTML / CSS / Bootstrp 簡介 / Jquery 簡介/ JAVASCRIPT 基礎 / HEXO 架站 / 還有簡單的 Node.js 介紹等等， 影音課程的部分其實到結訓前我也還沒全部看完，只看完了交作業需要用到的技術相關的影音課程部分，其餘的部分因為時間不足而暫時放棄觀看，不過這部分其實結束後還有蠻長的觀看時間可以慢慢去把影音課程的部分看完。另外就是影音課程的部分可以提交小節作業，這部分可以讓助教協助幫忙觀看你的程式碼部分有沒有哪邊需要改進的地方，算是體驗營內很超值的部分。

    2. 每日任務:
        每日任務是週一 ~ 週五每天都會有一個切版小題目和一個 Javascript 的小題目， 週五可能會出現較困難的題目作為週末讀物 毒物 ，在每日任務中其實包含了許多最終作業的拆解，有時候解完每日任務以後會有一種恍然大悟的感覺，原來這個地方可以用這種方式來處理，也可以讓你切實的感受到你有不斷累積，會的東西愈來愈多的感覺。另外還有一個最特別的地方是六角學院鼓勵你參考抄襲其他同學的答案，就算任務困難也不至於會有那種學習進度卡關的感受。

    3. 社群共學:
        社群共學包含了小組討論和 DC 專用頻道，由於學員眾多，其實可以感受到有很多認真上課的學員，同學間互助攻課每一道難題的感覺，也提升了參與的意願，很多問題也能快速獲得解答。

    4. 香香設計稿/Hexo 架站:
        參加這次軟體工程師體驗營很大一部份也是想拿香香設計稿，在課程剛開始的時候就有一種想法，是不是有辦法自己用 Hexo 去做出這個設計稿的內容，沒想到在最後竟然會直接提供設計稿的 theme 實在是很意外，非常超值。

    5. 作業:
        任務其實都有分難度，即使交最簡單的難度也可以通過，最終作業真的有一種集大成的感受，雖然大部分都在課程內容中。另外作業同樣會有助教進行批改，也可以觀摩其他同學的做法，讓你有一種好像做的出來卻又很痛苦的感受(?)

    6. 直播課:
        直播課的內容其實大部分都包含於影音課程，若有觀賞影音課程的話，其實有點像是在複習內容，不過也可以在直播中針對課程相關問題進行提問，也比較能了解到之前自己可能不太清楚或沒有注意到的地方。

    7. 送書:
        老師瘋狂在送書，光體驗營應該送了 50 本以上。

    以上這些內容只需要花費 799，雖然感受的到體驗營應該是希望推廣後續的課程給更多學生，但以體驗營來說，這樣的內容量在這樣的價格下真的很優惠也很辛苦。



    說完了體驗營好的部分，不免俗的也要來說一些可以加強的地方



    1. 小組/小組任務:
        雖然大家填表單的時候都說會參與，但實際上參與的人大多數仍然是在職生，所以會有一開始很熱絡但後繼無力的感受，除了因為課程後半部分愈來愈難外，可能另外一部份是因為第一週我的小組就有抽到書，抽到書後只能推派一位，以及後續重複組別無法重複領書等等的因素，因此大家參與的意願也愈來愈低。或許可以思考一些可以讓小組一直持續運作下去的方式也是不錯的選擇。

    2. 人數太多導致類似問題多:
        人數太多雖然參與的很熱烈，但管控上也會比較困難，因此常常也會有直播課的 QA 時間佔了一大部分，一直在詢問類似問題的現象，因此也常常下課時間已接近半夜 12 點左右的狀況。若是可以先把問題統整後再來一併回答或許可以比較節省時間。

    3. 可惡我沒有抽到書:
        太苦了。

        整體來說體驗營的部分絕對是很值得參加的，在這段期間內有確實的感受到自己學會的東西愈來愈多，也成功地架出了自己的 blog，明年若仍然有體驗營，我還是會參加來抽書。

    謝謝六角學院!! 謝謝校長!! 謝謝助教!!
